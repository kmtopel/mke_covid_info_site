Welcome to the MKE-COVID Resource Site!

This website is intended to function as a community forum wherein one can search for and post Milwaukee-area resources related to COVID-19 relief.

This site does not pretend to be an exhaustive list, and the accuracy of the posts can only be guaranteed insofar as the site administrator has time to verify links.

Please feel free to submit as many resources as you desire using the dropdown form in the link above. Do not hesitate to message the administrator via the contact link below to suggest any updates, revisions, or problems with the site.

Lastly, if you wish to contribute directly to the Developer of this site, you can do so through the support link at the bottom of the page.

Thank you!
